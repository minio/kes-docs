# Getting Started

1. Checkout or clone this repository

2. Create and checkout a new branch for your work

3. Download Hugo [0.105.0](https://github.com/gohugoio/hugo/releases/tag/v0.105.0)
 
   Refer to the install instructions for your OS in the [Hugo Getting Started Install Guide](https://gohugo.io/getting-started/installing/) to install from source.

4. Run `hugo server -D` from the `/source` directory to create the preview site

   View the site at `localhost:1313` or the location specified in the terminal output.

5. Make your edits as normal

   The live preview updates automatically as you save.

6. Commit changes with a detailed commit message, then push to the repository.
7. Open a pull request from your branch to `Main` to request a review of the changes.
   
   Once approved, the MinIO documentation team will merge your changes into the `Main` branch.
